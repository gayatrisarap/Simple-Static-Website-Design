    How to Fill MAH MCA CET 2021 Application Form?

    Candidates can apply by following the steps given below:
    Step 1:Registration
    1. After successful completion of registration process candidates would receive e-mails containing User ID and Password respectively. Use these login credentials to have access to the application form.
    2. Fill in the necessary details asked under separate heads
    3. Fill in the required details carefully to move ahead with the application process.
    4. Save and submit the details carefully after reviewing the application form.
    Step 2:Online Form Filling
    1. After successful completion of registration process candidates would receive e-mails containing User ID and Password respectively. Use these login credentials to have access to the application form.
    2. Fill in the necessary details asked under separate heads.
    3. Fill in the required details carefully to move ahead with the application process.
    4. Save and submit the details carefully after reviewing the application form.
    Step 3:Upload the Images
    In the next step, candidates would be required to upload their photograph and signature as a part of the registration process. It is important to adhere to the specifications mentioned by the authorities before uploading the scanned documents.

    Procedure for Uploading the Photograph and Signature

    1. Click on the respective link "Upload Photograph/Signature".
    2. Browse and select the location where the scanned photograph/signature file has been saved.
    3. Select the file.
    4. Click the 'Open/Upload' button.
    Step 4:Online Fee Payment
    • The application fee needs to be paid through online mode only.
    • The available option for remitting the fee are credit card/debit card, internet banking, IMPS, cash cards/mobile wallets, etc.
    • No other mode is available to pay fees
    • The application fee is non-refundable.
